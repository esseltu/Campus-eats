# Campus Eats

Campus Eats is a Progressive Web App (PWA) designed to help freshers and students at Central University (Miotso campus) find food spots on and near campus.

## Features

- **Interactive Map**: View food spots on a full-screen map.
- **Directions**: Get real-time directions to any food spot via Google Maps.
- **Open Status**: Instantly see if a spot is "Open Now" or "Closed" based on current time.
- **Filtering**: Filter spots by status, price, or type (e.g., Ghanaian, Cheap Eats).
- **PWA Support**: Installable on mobile devices for offline access.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Map**: React Leaflet, OpenStreetMap, CartoDB Voyager tiles
- **Backend**: Firebase Firestore (for real-time data)

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/esseltu/Campus-Eats.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
